Vue Fes Japan 2018
==================

[![CircleCI](https://circleci.com/gh/kazupon/vuefes-2018/tree/master.svg?style=svg)](https://circleci.com/gh/kazupon/vuefes-2018/tree/master)

## Setup

```
npm install
```

## Development

Serve with hot reload at localhost:3000

```
npm run dev
```

## Testing

```
npm test
```

## Generate

Generate static project.

```
npm run generate
```
